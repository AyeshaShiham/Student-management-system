# Student Management System

## Description
This is a Python-based Student Management System project. It allows users to:
- Add new students
- View student records
- Update student information
- Delete student records

## Features
- GUI interface using Tkinter
- Stores data locally (or can use database like MySQL)
- Simple and user-friendly interface
